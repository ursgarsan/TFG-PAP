getGrupos(curso): Esta función obtiene los grupos de un curso específico. Primero, realiza una búsqueda en la base de datos de los grupos asociados a ese curso. Luego, normaliza los datos de los grupos obtenidos, eliminando aquellos que no tengan una asignatura asociada, y formatea los datos de cada grupo para devolverlos de manera consistente.

getDocentes(curso): Similar a getGrupos, pero para obtener los docentes. Busca en la base de datos aquellos usuarios que no sean administradores, los cuales representan a los docentes. Luego, para cada docente, recupera sus solicitudes para el curso seleccionado, formateando los datos necesarios para el proceso de asignación de grupos.

esOnline(grupo): Esta función verifica si un grupo es online, es decir, si no tiene días ni horarios de clase definidos.

hayConflictoPorHorarioConGrupo(grupo1, grupo2): Comprueba si hay conflicto de horario entre dos grupos. Verifica si comparten días de clase y, en caso afirmativo, si los horarios se solapan.

hayConflictoPorHorarioConGrupos(gruposAsignados, grupo): Similar a la función anterior, pero verifica si hay conflictos de horario entre un grupo y un conjunto de grupos ya asignados a un docente.

superaLimites(docente, grupo): Comprueba si asignar un grupo a un docente supera los límites de créditos asignados, créditos restantes o el balance negativo permitido para un cuatrimestre.

seAlejaDeCeroCreditos(docente, grupo): Verifica si asignar un grupo a un docente hace que este se aleje demasiado de tener cero créditos restantes en el cuatrimestre.

hayConflictoPorCreditos(docente, grupo): Determina si hay conflictos de créditos al asignar un grupo a un docente, considerando los límites establecidos.

cumpleCondiciones(docente, grupo, asignaciones): Verifica si un docente cumple todas las condiciones para que se le pueda asignar un grupo, incluyendo la ausencia de conflictos de horario y de créditos.

asignarGrupo(grupo, docente, asignaciones): Asigna un grupo a un docente y actualiza los registros de créditos asignados y restantes para ese docente.

obtenerMejorDocenteParaGrupo(grupo, docentes, asignaciones): Esta función encuentra al mejor docente para asignar un grupo, teniendo en cuenta las restricciones de horario y créditos, así como también busca minimizar el desequilibrio de créditos acumulados entre los docentes.

obtenerMejorGrupoParaDocente(docente, grupos, asignaciones): Al contrario que la función anterior, esta busca el mejor grupo para asignar a un docente, teniendo en cuenta las mismas restricciones y priorizando los grupos con menos docentes asignados.

asignarGruposADocentes(grupos, docentes): Esta función asigna grupos a docentes. Primero, ordena los grupos y docentes en base a ciertos criterios, como el número de grupos/docentes ya asignados, y luego asigna grupos a docentes usando las funciones anteriores.

getCreditosAcumulados(docente, asignaciones): Calcula los créditos acumulados por un docente en función de las asignaciones realizadas.

imprimirAsignaciones(asignaciones): Imprime las asignaciones realizadas, mostrando los grupos asignados a cada docente y los créditos acumulados por cada uno.
